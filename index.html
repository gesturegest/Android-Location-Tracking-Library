Android Location tracking library

Android library with location tracking feature and many more

View project onGitHub

Track location every where and any where

The library can be used to develop location tracking application, It is fully customized and has features of GCM(Google cloud messaging) support. This library allow you to develop any type of tracking application for every version of android app .It has features for communication with server asynchronously, sending user registration data,encryption decryption helper class, utility helper class etc.
